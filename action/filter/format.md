# 格式制定

* 條件格式:

  * source:  \(String\)  // 從哪個工具發出

    type:    \(String\)   // 兩種模式: ‘equal’, ‘range’

    content: \(array\)  // 篩選條件的標題名稱

    val:   \(Array\)  // 條件的值，用前述提到的tag來分類

         tag1: \(Array\)

         tag2: \(Array\)

         ……

* 例: 後分類篩選 選區為第一選區\(士林北投\)的男性的資料
